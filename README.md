## Использование

1. **Включи пакет:** `PIDOR / Enable Package`.
2. **Выделение объектов на сцене:** Alt + клик на объект → выбираем цвет.
3. **Выделение папок:** Alt + клик на папку → выбираем цвет.
4. **Избранное:** Drag&Drop ассетов в окно `Favorites Window` (`PIDOR / Open Favorites`).
5. **Удаление элементов:** кнопка `X` рядом с объектом в избранном.
6. **Переключение сцены:** кнопка `Scene` рядом с кнопками запуска playmode.
7. **Переключение состояние объекта в иерархии:** кнопка `A` при наведении на объект в иерархии.

---

## Поддержка
- Работает только в **Unity Editor**.
- Локальные настройки сохраняются для каждого пользователя отдельно.
- Совместимо с Unity 2021+.
